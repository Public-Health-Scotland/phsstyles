# Changelog

## phsstyles 0.1.2 (2024-08-06)

- Exported `phs_colour_values`, `phs_palettes` and `phs_palette_types`.
  This now allows users to call phsstyles functions without needing to
  load the package.

## phsstyles 0.1.1 (2022-05-30)

- Make axis title not blank in
  [`theme_phs()`](https://public-health-scotland.github.io/phsstyles/reference/theme_phs.md)
  ([\#3](https://github.com/Public-Health-Scotland/phsstyles/issues/3)).

- Add alias for function
  [`phs_colours()`](https://public-health-scotland.github.io/phsstyles/reference/phs_colours.md),
  [`scale_colour_discrete_phs()`](https://public-health-scotland.github.io/phsstyles/reference/scale_colour_discrete_phs.md)
  and
  [`scale_colour_continuous_phs()`](https://public-health-scotland.github.io/phsstyles/reference/scale_colour_continuous_phs.md)
  so that users can use American spelling as “color”.

- Update the documentation for `phs_palettes` and `phs_palette_types`.

- Create a [pkgdown
  webpage](https://public-health-scotland.github.io/phsstyles/) so that
  readme, help pages and news page are integrated.

## phsstyles 0.1.0 (2021-10-15)

- Initial package release.
- [`phs_colours()`](https://public-health-scotland.github.io/phsstyles/reference/phs_colours.md),
  [`theme_phs()`](https://public-health-scotland.github.io/phsstyles/reference/theme_phs.md),
  [`scale_colour_discrete_phs()`](https://public-health-scotland.github.io/phsstyles/reference/scale_colour_discrete_phs.md),
  [`scale_colour_continuous_phs()`](https://public-health-scotland.github.io/phsstyles/reference/scale_colour_continuous_phs.md),
  [`scale_fill_discrete_phs()`](https://public-health-scotland.github.io/phsstyles/reference/scale_fill_discrete_phs.md)
  and
  [`scale_fill_continuous_phs()`](https://public-health-scotland.github.io/phsstyles/reference/scale_fill_continuous_phs.md)
  functions added.
